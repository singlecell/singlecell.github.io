# singlecell.github.io

Personal academic homepage of **Linnan Li** — pharmaceutical analysis, mass spectrometry, and metabolomics.

🔗 Live site: https://singlecell.github.io

Built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme (MIT). Edit content in:

- `_pages/about.md` — bio
- `_bibliography/papers.bib` — publications
- `_data/cv.yml` — CV
- `_news/` — news items
- `_config.yml` / `_data/socials.yml` — site config and links

Pushing to `main` auto-rebuilds and deploys the site via GitHub Actions.
